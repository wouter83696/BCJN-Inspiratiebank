# Activiteitafbeeldingen

Deze map is bedoeld voor beelden die op de inspiratietegels kunnen verschijnen.

- Vervang de voorbeeldfoto's gerust door eigen `.webp`, `.jpg` of `.png` bestanden.
- Gebruik bij voorkeur liggende beelden rond 1200 pixels breed.
- Maak voor vaste JPG-foto's ook varianten met `-480.jpg` en `-800.jpg`; de publieke site gebruikt die automatisch via `srcset`.
- Houd bestanden klein, liefst onder 250 KB per afbeelding.
- Foto's die via het inspiratieformulier worden meegestuurd, worden eerst opgeslagen als `pending`.
- Een ingestuurde foto verschijnt pas op een tegel wanneer `imageStatus` op `approved` staat.
